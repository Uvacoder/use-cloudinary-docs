---
title: useGif
---
The `useGif` hook gives you the option to call either a video as a looping gif, or a gif stored in your account. The result is a URL generated by the SDK. 

```jsx
import { useGif } from 'use-cloudinary'

function Gif({ cloudName, publicId, transforms }) {
  const {getGif, data, status, error} = useGif({ cloud_name: cloudName })

  React.useEffect(() => {
    getGif({
      public_id: publicId,
      transform_options: {
        ...transforms
      }
    })
  }, [])

  if (status === "loading") return <p>Loading...</p>;
  if (status === "error") return <p>{error.message}</p>;

  return <img src={data} alt='gif from a video'/>
}
```

## getGif()

The `getGif` function is just a `setState`, used to fire off our internal query and pass our options into our SDK to generate our URL.
