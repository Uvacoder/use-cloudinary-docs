---
title: useVideo
---
The `useVideo` hook gives you the ability to call an image from your Cloudinary account and apply transformations. The result is a URL generated by the SDK. 

```jsx
import { useVideo } from 'use-cloudinary';

function Video({ cloudName, publicId, transforms }) {
  const { getVideo, data, status, error } = useVideo({ cloud_name: cloudName });
  
  React.useEffect(() => {
    getVideo({
      public_id: publicId,
      transform_options: {
        ...transforms
      }
    })
  }, [])

  if (status === "loading") return <p>Loading...</p>;
  if (status === "error") return <p>{error.message}</p>;

  return (
    <video autoPlay controls>
      <source src={data} />
    </video>
  )
}
```

## getVideo()

The `getVideo` function is just a `setState`, used to fire off our internal query and pass our options into our SDK to generate our URL.
